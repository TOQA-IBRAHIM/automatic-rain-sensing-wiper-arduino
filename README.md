# Automatic Rain Sensing Wiper System using Arduino

## 📌 Overview
This project presents an automatic rain-sensing wiper system designed to improve driving safety by eliminating the need for manual control.

The system detects rain intensity and automatically activates the wiper mechanism using an Arduino-based embedded system.

---

## 🎯 Objective
To design and implement an embedded system that automatically controls car wipers based on rain detection.

---

## 🚨 Problem
Traditional car wipers require manual activation, which can distract drivers and reduce safety during sudden rainfall.

---

## 💡 Solution
This system uses a rain sensor to detect water and automatically control wiper movement through servo motors.

When rain is detected:
- The system activates the wipers  
- The wiper speed can adjust based on rain intensity  
- The system stops automatically when no rain is detected  

---

## ⚙️ System Components
- Arduino UNO (Microcontroller)  
- Rain Detection Sensor  
- Servo Motors  
- Breadboard and circuit connections  

---

## 🔄 System Workflow
1. Rain sensor detects water on its surface  
2. Signal is sent to the Arduino  
3. Arduino processes the input using programmed logic  
4. Servo motors are activated to simulate wiper movement
